# Share SmartChef With Customers

SmartChef is now packaged as a customer-facing web app.

## The Simple Version
Upload this folder to a static hosting service and send customers the public link.

Good beginner-friendly options:
- Netlify Drop.
- Vercel.
- Cloudflare Pages.
- GitHub Pages.

## What Customers Can Do
- Complete onboarding.
- Choose cuisine, diet, lifestyle, and cooking time.
- Add inventory with starter packs, search, voice-style text, and scan simulation.
- View meal recommendations.
- Open recipe details.
- Try weekly planning and Premium pricing.
- Leave feedback in the Feedback tab.
- Copy their feedback summary and send it to you.

## Important Limitation
Right now, feedback is saved on the customer's own phone or browser. It does not automatically come to you because there is no backend database connected yet.

For early testing, ask customers to tap **Copy feedback summary** and send it to you through WhatsApp, email, Instagram, or any channel you prefer.

## When To Add A Backend
Add a backend after customers prove they want the app.

Backend features to add later:
- Real user accounts.
- Feedback submitted directly to you.
- Real AI recipe generation.
- Real photo and receipt scanning.
- Cloud inventory sync.
- Subscription payments.

## Suggested First Test
Send it to 10-20 people and ask:

"Please use this like a real cooking app. Then go to the Feedback tab, rate it, and send me the copied feedback."

If several people say they would use it weekly, the next step is building the real MVP backend.
