# Publish SmartChef To GitHub Pages

## What This Folder Is
This folder is ready to be published from the root of the `main` branch.

## GitHub Pages Settings
After pushing to GitHub:
1. Open the GitHub repository.
2. Go to **Settings**.
3. Go to **Pages**.
4. Under **Build and deployment**, choose:
   - Source: **Deploy from a branch**
   - Branch: **main**
   - Folder: **/ root**
5. Save.

GitHub will give you a public URL after it finishes publishing.

## Customer Link
Send customers the GitHub Pages URL. They can open it on a phone and test SmartChef like a real app.
